# expense-service (Persistence Start)

Este proyecto es el **punto de partida** del laboratorio de persistencia con Spring Boot 4.

Sigue las instrucciones en [LAB 4 - SPRING BOOT PERSISTENT.md](../LAB%204%20-%20SPRING%20BOOT%20PERSISTENT.md) para completar el laboratorio.

## Tecnologías

- Spring Boot 4.0.2
- Spring MVC
- springdoc-openapi (Swagger UI)

## Lo que vas a agregar

- Spring Data JPA
- H2 Database
- Bean Validation
- Entidades JPA con relaciones
- Repositorios Spring Data
- Servicio transaccional
- Paginación y ordenamiento

## Ejecutar (después de completar el lab)

```bash
cd expense-service
mvn spring-boot:run
```
