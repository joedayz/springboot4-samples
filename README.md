# springboot4-samples
Curso de Spring Boot 4 inspirado en `quarkus-bcp-2025`, con enfasis en las grandes diferencias
entre ambos ecosistemas. Este repositorio contiene el temario oficial y la estructura de
modulos para construir los mismos temas con Spring Boot 4.

## Que encontraras aqui
- `curso/temario.md`: temario completo con duraciones y hands-on.
- `curso/README.md`: estructura del curso, convenciones y flujo sugerido.
- `curso/diferencias-quarkus-vs-springboot4.md`: comparativa principal por areas.
- `curso/modulos/`: guias por modulo con objetivos y hands-on.

## Como usar este repo
1. Lee `curso/temario.md` para la vision general.
2. Revisa `curso/diferencias-quarkus-vs-springboot4.md` para la comparativa honesta.
3. Cada modulo ira incorporando proyectos `*-start` y `*-solution` con ejercicios y soluciones.

Si quieres que agregue el codigo de los modulos, indicalo y lo priorizo por bloque (por ejemplo,
REST, Persistencia, Seguridad).
