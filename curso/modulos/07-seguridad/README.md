# 07 - Seguridad con Spring Security (2 horas)

## Objetivo
Aplicar seguridad moderna stateless con JWT y OAuth2/OIDC.

## Contenido
- Spring Security moderno (sin XML)
- JWT
- OAuth2 / OpenID Connect
- Integracion con Keycloak / Auth0

## Hands-on
- Proteger endpoints.
- Roles y scopes.
- Seguridad stateless.

## Diferencias clave vs Quarkus
- Configuracion basada en beans vs SmallRye OIDC/JWT.
- Filtros y `SecurityFilterChain` en Spring.
