# 01 - Configuracion externa y perfiles (1 hora)

## Objetivo
Dominar configuracion por ambientes en Spring Boot, incluyendo overrides por ENV y ConfigMap.

## Contenido
- `application.yml` y perfiles
- Variables de entorno
- Configuracion por ambiente
- `@ConfigurationProperties`
- Spring Cloud Config (conceptos)

## Hands-on
- Configuracion por dev, test, prod.
- Override via ENV y ConfigMap en Kubernetes.

## Diferencias clave vs Quarkus
- `spring.profiles.active` vs `quarkus.profile`.
- `@ConfigurationProperties` y binding avanzado en Spring.
