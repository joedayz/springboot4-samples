# 08 - Implementacion en Kubernetes (2 horas)

## Objetivo
Empaquetar y desplegar microservicios Spring Boot en Kubernetes.

## Contenido
- Dockerizacion de Spring Boot
- Liveness / Readiness probes
- ConfigMaps y Secrets
- Despliegue en Kubernetes

## Hands-on
- Build Docker.
- Deploy en K8s.
- Acceso via Service.

## Diferencias clave vs Quarkus
- Layered jars y buildpacks en Spring.
- Probes via Actuator endpoints.
