# 00 - Introduccion (1 hora)

## Objetivo
Dar el contexto cloud-native y la comparativa honesta entre Spring Boot 4 y Quarkus,
y dejar un microservicio basico corriendo.

## Contenido
- Que significa cloud-native hoy
- Evolucion de Spring → Boot 3 → Boot 4
- Monolito vs microservicios
- JVM tradicional vs Native Image
- Spring Boot vs Quarkus (comparativa honesta)
- Arquitectura del curso

## Hands-on
- Crear proyecto con Spring Initializr.
- Estructura basica.
- Primer endpoint REST.

## Diferencias clave vs Quarkus
- Spring Boot usa starters y auto-configuracion; Quarkus usa extensiones build-time.
- Dev mode: DevTools vs `quarkus:dev` con Dev UI.
