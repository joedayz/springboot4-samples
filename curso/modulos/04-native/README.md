# 04 - Aplicaciones nativas con Spring Boot y GraalVM (2 horas)

## Objetivo
Entender el modelo AOT de Spring Boot 4 y construir una aplicacion nativa.

## Contenido
- GraalVM Native Image
- AOT en Spring Boot 4
- Diferencias JVM vs Native
- Limitaciones reales

## Hands-on
- Compilar app nativa.
- Comparar tiempos de arranque y memoria.
- Docker + Native Image.

## Diferencias clave vs Quarkus
- Quarkus es nativo-first; Spring requiere mas ajustes de reflection.
- AOT en Spring Boot 4 mejora tiempos, pero no elimina todas las restricciones.
