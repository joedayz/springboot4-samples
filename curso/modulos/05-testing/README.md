# 05 - Testing en microservicios (2 horas)

## Objetivo
Establecer una estrategia de testing completa para microservicios Spring Boot.

## Contenido
- `@SpringBootTest`
- MockMvc / WebTestClient
- Testcontainers
- Tests de integracion reales

## Hands-on
- Test REST.
- Test con base de datos real (Testcontainers).
- Test en pipeline.

## Diferencias clave vs Quarkus
- Slices (`@WebMvcTest`, `@DataJpaTest`) disponibles en Spring.
- Integracion nativa con Testcontainers en Spring Boot.
