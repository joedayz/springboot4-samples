# 03 - Persistencia (2 horas)

## Objetivo
Aplicar Spring Data JPA y transacciones, con una introduccion a R2DBC.

## Contenido
- Spring Data JPA
- Hibernate 6
- Mapeo de entidades
- Transacciones
- Introduccion a R2DBC (reactivo)

## Hands-on
- CRUD con JPA.
- Query methods.
- Paginacion.

## Diferencias clave vs Quarkus
- Repositorios Spring Data vs Panache.
- Configuracion de transacciones via `@Transactional`.
